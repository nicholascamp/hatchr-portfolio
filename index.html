Did's Hostel Website responsivo

Trazer para internet todo o espírito que o pessoal do Did's Hostel criou em seu ambiente foi o nosso maior desafio nesse projeto. Como criar um ambiente online tão agradável e receptivo, fazendo com que os usuários se sintam no próprio hostel?

A partir da parceria com o designer Koji Takaki, nós partimos para tentar elevar ao máximo a experiência dos usuários, afinal eles tem que se sentir tão bem no site como no hostel. Além disso, precisávamos levar em consideração que muitos usuários seriam viajantes, sem um acesso rápido de internet e muitas vezes no celular, tentando achar o melhor caminho para o Did's Hostel. Para isso nós criamos um website 100% responsivo, cuidando para que cada detalhe funcionasse da maneira correta em cada dispositivo.

Na segunda etapa do projeto, nós desenvolvemos uma forma dos usuários realizarem a reserva de suas camas no próprio site do Did's Hostel. Para facilitar a vida deles, integramos com o sistema de booking que eles já tinham, assim todas as informações rodam em tempo real entre os usuários e o Did's.

Star Uniforms Sistema de Automação

Com o objetivo de se posicionar como a empresa de confecção de uniformes mais inovadora do mercado, a Star Think nos propôs o desafio de construir uma plataforma online em que ela pudesse se relacionar de uma forma inteligente com seus clientes.

Para isso, nós criamos o Sistema Integrado de Gestão de Pedidos, uma plataforma web que permite a gestão dos pedidos de uniformes, estoques e informações estratégicas dos colaboradores das empresas parceiras da Star Think Uniforms.

Foi muito gratificante para nós saber que nosso sistema pôde contribuir efetivamente com a evolução da empresa dentro do mercado. Através dessa automatização de processos e a possibilidade de acessar o sistema de qualquer dispositivo conectado a internet, a empresa poderá gerenciar de maneira eficaz as principais atividades decisórias envolvidas no seu negócio.

Recruta Arpex Website Responsivo

Em alguns momentos temos que ser práticos e objetivos para gerar resultados de forma ágil. Essa é a frase que resume o projeto do site de recrutamento da Arpex Capital, o maior grupo de empresas de pagamentos eletrônicos do Brasil.

Precisávamos desenvolver uma plataforma simples e direta para que os novos recrutas conseguissem conhecer todo o processo e se candidatarem de forma rápida. Pelo fato da Arpex já ter os dados de todos os processos seletivos armazenados no TypeForm, nós fizemos uma integração através da API deles para que eles pudessem continuar a usar esse serviço no site novo.

Além disso, foi necessária muita pesquisa, trabalhos e testes para conseguir um site que cumprisse o seu papel: Aumentar a conversão de novos recrutas. Nesse sentido, orientamos toda a experiência do usuário para que ele sempre tenha a cada etapa do site uma forma fácil de ir para o formulário.

recrutaarpex.com.br

Allpet Plataforma Geolocalização - App Mobile

A AllPet é uma startup de geolocalização de serviços de Pet. A participação da Hatchr no projeto começou com o desenvolvimento dos wireframes e prosseguiu até a entrega do MVP.

De um lado, a startup AllPet precisava de uma ferramenta online que permitisse aos usuários localizar facilmente os serviços para seus animais de estimação mais próximos. Para isso desenvolvemos funções simples e objetivas que permitem atingir os objetivos de forma intuitiva.

Do outro lado, precisávamos que os prestadores de serviço pudessem se cadastrar na plataforma, inserindo assim informações que fossem relevantes na hora da busca. Além disso, também desenvolvemos um painel de controle para que os gestores da startup AllPet possam gerenciar todo o seu negócio de forma prática. O desenvolvimento de aplicações e MVP's para startups é um dos pilares da Hatchr e esse é um grande exemplo de trabalho que nos motiva a continuar nesse caminho.
